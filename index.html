Sistema de gestión para restaurante con cajero automático

Descripcion

Proyecto desarrollado en NetBeans con Java y POO que simula el funcionamiento de un cajero automático para un restaurante. Permite el registro e inicio de sesión de empleados para acceder a una interfaz donde se pueden añadir platillos mediante formularios y gestionar las mesas ocupadas a través de botones interactivos. Toda la información registrada se almacena y se visualiza en una tabla para control interno del restaurante.

Sistema de parqueadero con control de celdas

Descripcion

Proyecto en Java y POO desarrollado en NetBeans que gestiona el ingreso y salida de vehículos en un parqueadero. Existe un sistema de login y registro para los empleados. El parqueadero cuenta con celdas que representan el estado (vacio o ocupado) por medio de colores. Al registrar un vehículo se abre un formulario dedicado y la información relacionada se almacena y muestra en una tabla, facilitando el seguimiento del parqueadero en tiempo real.
